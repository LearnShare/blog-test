# Blog

A blog system.

- Monorepo - Turborepo
- DB - Prisma
    - PostgreSQL
    - Docker
- API - Express
    - dotenv - .env 配置文件
    - zod - 数据格式验证
    - bcrypt - 密码 hash 和验证
    - jsonwebtoken - 登录和身份验证
- UI - Next.js
    - Tailwind.css
    - shadcn/ui
- 其它工具
    - Tabby
    - OrbStack
    - VS Code
    - Chrome
    - API 文档（Open API）
